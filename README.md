# wedding

## About

結婚式の招待状アプリ

## 開発

開発環境の構築手順

### prerequisit

. [Node.js](https://nodejs.org/ja)

### ローカル起動手順

```
cd front
npm install
npm run dev
```

## デプロイ

デプロイ手順
tagを作成する
例
```
git tag front_20230709_01
git push origin front_20230709_01
```
